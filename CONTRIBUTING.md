# Contribuyendo a Tarjeta-Digital

Gracias por tu interés en mejorar este proyecto. 

### Reglas de Oro:
1. **Simplicidad ante todo**: Si una función no es crítica, no se agrega.
2. **Cero Dependencias**: No se aceptan frameworks pesados. Todo es Vanilla.
3. **Optimización**: Cada línea de CSS debe justificarse por su eficiencia en recursos.

Para proponer cambios, abre un "Pull Request" con una explicación técnica de la mejora.
